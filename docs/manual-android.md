# Android 向け使用マニュアル
## Hono MiniForge — Novel Game Forge

**対象端末:** Android スマートフォン（Pixel 6a など）  
**必要なもの:** ブラウザ（Chrome 推奨）のみ / アプリインストール不要

---

## はじめに

このサービスはブラウザだけで使えます。  
ローカルサーバーの起動やアプリのインストールは**不要**です。

---

## 1. アクセス方法

Chrome（または任意のブラウザ）で以下の URL を開いてください。

| ページ | URL |
|--------|-----|
| サーバートップ | `https://xxx.vercel.app/` |
| 生成シート | `https://xxx.vercel.app/sheet.html` |
| マニュアル | `https://xxx.vercel.app/manual.html` |

> `xxx` の部分は実際の Vercel プロジェクト名に置き換えてください。

**ホーム画面に追加する方法（Android Chrome）**

1. Chrome で URL を開く
2. 右上メニュー（︙）→「ホーム画面に追加」
3. 名前を付けて「追加」→ アプリのように起動できます

---

## 2. 生成シートの使い方

`sheet.html` を開いて、上から順に項目を埋めていきます。

### 各セクションの入力内容

#### 📋 Project Info
| 項目 | 内容 |
|------|------|
| サムネイル | タップして画像を選択（任意） |
| 表示名 | ゲームのタイトル（日本語OK）|
| ファイルID | 英数字・アンダースコアのみ（例: `my_game_01`） |
| 概要メモ | 一言説明（任意） |

#### 🌍 World（世界観）
- **世界名**: ゲーム世界の名称
- **文明レベル**: 中世・魔法・SF など
- **支配構造**: 誰が世界を支配しているか
- **禁忌**: この世界でやってはいけないこと
- **世界の問題**: 解決すべき大きな課題

#### 👤 Characters（登場人物）
- **主人公**: 名前・目的・恐怖・欠点・戦う理由
- **ヒロイン/重要人物**: 名前・主人公との関係・隠し事

#### ⚙️ Game Systems
タップしてゲームに必要なシステムを選択（複数選択可）

| システム | 用途 |
|---------|------|
| 分岐 | 選択肢によってストーリーが変わる |
| 好感度 | キャラクターとの関係値管理 |
| 探索 | マップや場所を調べる機能 |
| アイテム | 道具の収集・使用 |
| 戦闘 | バトルシステム |
| 用語辞典 | 世界観用語の説明表示 |
| CG収集 | イベント画像の収集 |
| マルチエンド | 複数のエンディング |

#### 📖 Scenario Intro（シナリオ導入）
- **最初の事件**: ゲームが始まるきっかけ
- **プレイヤーの目的**: ゲーム全体の目標
- **第一章開始イベント**: 最初に起きる具体的な場面

## 3. JSON Export の手順

1. 各項目を入力する
2. 画面下部の **「⬇ JSON Export」** ボタンをタップ
3. ファイルが自動でダウンロードされます（例: `my_game_01_sheet.json`）
4. 画面下部の **API Preview** 欄に JSON の内容が表示されます

> ダウンロードされたファイルはブラウザの「ダウンロード」フォルダに保存されます。

---

## 4. ファイルの保存方法（Dropbox）

1. Files アプリ（またはファイルマネージャー）を開く
2. ダウンロードフォルダから JSON ファイルを見つける
3. Dropbox フォルダへ移動またはコピー

---

## 5. API テスト画面の使い方

`sheet.html` のヘッダーにある **「🔌 API テスト」** リンクから移動できます。  
（または `100_gamecollection` アプリの HONO_API テスト画面）

### 基本操作
1. 右上の **Base URL** を Vercel URL に変更する
2. 左サイドバーからエンドポイントを選択
3. **「▶ Send」** を押してレスポンスを確認

### パズルゲームのプレイ手順
1. `② パズル初期化` を実行 → `sess` が自動取得される
2. `③ 操作送信` で `move_index` を指定してタイルを動かす
3. `④ 解答判定` でクリア確認

## 6. よくある問題と解決策

| 症状 | 原因 | 解決策 |
|------|------|--------|
| ページが開かない | URL が違う | Vercel URL を再確認 |
| 「取得に失敗」が出る | Base URL が localhost になっている | Vercel URL に変更 |
| JSON がダウンロードされない | ブラウザの設定 | ダウンロードを許可する |
| 画像が選択できない | ブラウザの権限 | ストレージへのアクセスを許可 |
| 入力が消えた | ページをリロードした | ページを閉じる前に Export する |

> **注意:** 入力内容はブラウザを閉じると消えます。必ず Export してから閉じてください。
